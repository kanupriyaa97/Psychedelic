1. Install panda3d from http://www.panda3d.org/download/panda3d-1.9.2/Panda3D-SDK-1.9.2.exe

2. Copy 
		"Models", "Game.py" in Psychedilic Folder to root of 
		panda3d installation directory
		
		eg. if panda is installed in C:/Panda3D-1.9.2
			paste in C:/Panda3D-1.9.2/


		IF ANY OPTIONS SHOW UP , PLEASE MERGE FOLDERS WITH SAME FOLDER NAME

3. Press Ctrl+Shift and Right Click on empty space in file explorer 
	Select "Open Command Prompt Here"

4. In command prompt , type ppython game.py
		(ppython initialises a special version of panda3d python)
